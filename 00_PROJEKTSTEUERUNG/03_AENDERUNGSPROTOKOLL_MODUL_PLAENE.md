# Änderungsprotokoll – BBM-Modul „Pläne“

Dieses Dokument hält jede gemeinsam beschlossene Abweichung vom Zielbild oder Meilensteinplan fest.

## Verbindliche Regel

Keine Richtungsänderung erfolgt stillschweigend. Codex darf Zielbild, Reihenfolge, Umfang oder technische Grundentscheidung nicht eigenmächtig ändern.

## Vorlage

```text
Datum:
Beschlossen von:
Betroffener Meilenstein:
Bisheriger Plan:
Neue Entscheidung:
Begründung:
Auswirkung auf bestehende Arbeit:
Auswirkung auf spätere Meilensteine:
Neue Abnahmekriterien:
Status:
```
